BookCompass プライバシーポリシー

GrowthLab（以下「当方」）は、BookCompass（以下「本アプリ」）で取り扱う情報について、以下のとおり定めます。なお、現在のリリース版の本アプリは会員登録なしで利用でき、アプリ内データは匿名ユーザーIDに紐づいて管理されます。

1. 取得する情報

本アプリでは、以下の情報を取り扱います。

  • 識別情報（匿名ユーザーID、サブスクリプション状態の確認に必要な識別子など）
  • 読書記録（書籍のタイトル・著者・ISBN・ジャンル・表紙画像URL・読書ステータス・読了日など）
  • 呟き（本についてユーザーが入力した気づきやメモのテキスト）
  • XP・称号などのゲーミフィケーションデータ
  • 購入・サブスクリプション情報（課金状態・プラン種別など）
  • お問い合わせ時にユーザーが任意に送信する情報

将来、会員登録機能を追加した場合は、メールアドレスやApple / Google などの認証情報を取得することがあります。その際は本ポリシーを更新します。

2. 利用目的

取得した情報は、以下の目的で利用します。

  • 読書記録・呟き機能、知識レーダーチャート機能の提供
  • 呟きおよび本の特性のAI分析機能（書籍インサイト・読書傾向の可視化）の提供
  • サブスクリプション機能（Compass Pro）の提供
  • お問い合わせ対応
  • 品質改善および不具合調査

3. 保存先と外部送信

本アプリはクラウドバックエンドを利用しており、以下の外部サービスにデータを送信します。

  • Supabase（米国・EUなど）: ユーザー認証情報・読書記録・呟きなどのアプリデータを保存します。
  • RevenueCat（米国）: サブスクリプション管理のため、購入状態・デバイス識別子などを送信します。
  • AI処理サービス（現在はGoogle Gemini等を利用）: AI分析機能のため、呟きテキストをSupabase Edge Functionsを経由してAIサービスのAPIに送信します。ユーザーのデバイスから直接送信されるものではありません。なお、当方は利用するAIサービスに対してデータをAIモデルのトレーニングに使用しないよう設定しています。送信されたデータがAI学習に利用されることはありません。利用するAIサービスは変更される場合があります。
  • OpenBD API / 国立国会図書館サーチAPI: 書籍メタデータ取得のため、ISBN等の書籍情報を送信します。個人情報は含みません。
  • 課金処理にはAppleのStoreKitを利用します。

現在のリリース版では会員登録機能が未提供のため、アプリ削除や機種変更を行った場合、同じデータに再アクセスできなくなることがあります。引き継ぎに対応する会員登録機能は、リリース後の優先対応として追加予定です。

4. 第三者提供

当方は、法令に基づく場合を除き、個人情報を第三者に提供しません。
ただし、上記「3. 保存先と外部送信」に記載のサービス（Supabase・RevenueCat・AIサービス・Apple）には、サービス提供に必要な範囲でデータを委託します。

5. ユーザーによる管理

  • ユーザーはアプリ内で呟きの個別削除、本の記録削除を行えます。
  • 現在のリリース版では会員登録機能が未提供のため、機種変更時のデータ引き継ぎには対応していません。
  • アプリを端末から削除した場合、保存済みデータがサーバーに残っていても、同じデータへ再アクセスできなくなることがあります。
  • アカウントおよびサーバー上のデータの削除を希望される場合は、下記お問い合わせ先までご連絡ください。
  • アプリを端末から削除しても、Supabaseに保存されたデータは削除手続きが完了するまで残る場合があります。

6. 改定

本ポリシーは必要に応じて改定することがあります。
重要な変更はアプリ内表示または公開先で通知します。
